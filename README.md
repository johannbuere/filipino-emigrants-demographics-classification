# filipino-emigrants-demographics-classification

A compact data mining project that predicts the educational attainment of Filipino emigrants using demographic and socioeconomic features. The analysis is implemented in a single Jupyter notebook with end-to-end steps from data loading and preprocessing to model training, evaluation, and interpretation.

## Notebook

- **emigration_educ_classification.ipynb**: Exploratory analysis, feature selection, training of five classifiers, and performance comparison.

## Dataset

- **Source**: Kaggle - [Demographics of Filipino Emmigrants 2015-2020](https://www.kaggle.com/datasets/ivyfiecas/demographics-of-filipino-emmigrants-2015-2020/data)
- **Description**: Demographic and socioeconomic attributes (age, gender, civil status, education, occupation, and location) for emigration records, intended for trend analysis and predictive modeling. The dataset contains no personally identifiable information.
